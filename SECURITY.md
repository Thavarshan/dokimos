# Security Policy

## Supported Versions

Dokimos currently supports the latest code on the default branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for suspected security vulnerabilities.

Instead, contact the maintainer directly through GitHub at:

- https://github.com/Thavarshan

Include a clear description of the issue, reproduction steps if possible, and
any affected files or commands.

## Response Expectations

The project will aim to acknowledge security reports promptly and follow up with
status updates as triage progresses.
