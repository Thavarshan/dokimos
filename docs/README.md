# Dokimos Documentation

This directory contains the long-form project documentation for Dokimos.

It is structured to work as an mdBook and to remain readable directly on GitHub.

## Local Preview

If `mdbook` is installed locally, preview the docs with:

```bash
mdbook serve docs
```

Then open `http://localhost:3000`.
